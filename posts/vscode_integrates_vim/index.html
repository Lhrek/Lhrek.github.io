vim in vscode

1、不能降低vscode预先的体验

操作 vscode下常用快捷键 vim下该组合键的作用 切换到vim下的解决方法
选中当前单词修改 ctrl+d+修改 向下翻半页 ciw+修改
向上/下复制行 shift+alt+↑/↓ 无 yy/Y复制行 p向下复制 P向上复制
批量修改匹配项 ctrl+f2+修改 无 保留ctrl_f2_c，vim下是 :%s/old/new/g //todo选中后如何修改
全选 ctrl+a 选中数字加1 保留
复制 ctrl+c esc 保留ctrl_c,因为vim的y复制并没有打通系统剪切板，如果需要将文本复制到vim之外的系统，需要配置完使用长指令"+y，不方便
粘贴 ctrl+v 进入可视模式 保留ctrl_v
查找 ctrl+f 向下滚一页 保留ctrl_f
关闭侧边栏 ctrl+b 向上滚一页 保留ctrl_b
关闭当前页面 ctrl+f4/ctrl+w ctrl_w再加一个命令可以执行特定的窗口操作 ctrl_w_c/ctrl_f4 
    "vim.": {
        "<C-a>": false,
        "<C-b>": false,
        "<C-c>": false,
        "<C-f>": false,
        "<C-s>": false,
        "<C-v>": false,
        "<C-z>": false
    },//取消vim中的部分键位映射，使用vscode的
    "vim.normalModeKeyBindings": [ //自定义键位（空格加jk翻页）
        {
            "before": ["<leader>", "j"],
            "after": ["<C-f>"]
        },
        {
            "before": ["<leader>", "k"],
            "after": ["<C-b>"]
        }
    ],
    "vim.leader": "<space>"  //修改leader键为空格

使用vimtutor练习

A [Appending 在行尾输入]

:x [:wq的缩写 保存并退出编辑器]

x [普通模式下的删除光标下的字符]

dw/d$/de [d-删除;w-until the start of the next word ;e-to the end of the current word;$-to the end of the line]

0 [start of the line]

dd/ndd [delete n lines,and store it in a Vim register ]

u [undo the last commands]

U [fix a whole line]

ctrl + r [like vscode ctrl+y,To undo the undo']

p [put the line below the cursor]

​ c/d [number] motion

rx [replace the character at the cursor with x]

ce [chang until the end of a word]

c$ [end of line]

nG/:n [move to the NTH row in the file]

gg [move to start of file]

G [move to the bottom of file]

/ ？ [search]

% [match ({[]}) ]

%s/old/new/g [ to change every occurrence in the whole file.]

type v motion :w FILENAME [To save part of the file]

o [open a line below the cursor and place you in Insert mode]

e [jump at the enf of words ]

a [appending at the end of word]

R [replace more than one character`]
