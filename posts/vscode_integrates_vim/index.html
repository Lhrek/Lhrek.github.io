vim in vscode

操作 vscode下常用快捷键 切换到vim下的解决方法
选中当前单词修改 ctrl+d+修改
向上/下复制行 shift+alt+↑/↓
批量修改匹配项 ctrl+f2+修改
全选 ctrl+a
复制 ctrl+c
粘贴 ctrl+v
查找 ctrl+f
关闭侧边栏 ctrl+b
关闭当前页面 ctrl+f4/ctrl+w

使用vimtutor练习

A [Appending 在行尾输入]

:x [:wq的缩写 保存并退出编辑器]

x [普通模式下的删除光标下的字符]

dw/d$/de [d-删除;w-until the start of the next word ;e-to the end of the current word;$-to the end of the line]

0 [start of the line]

dd/ndd [delete n lines,and store it in a Vim register ]

u [undo the last commands]

U [fix a whole line]

ctrl + r [like vscode ctrl+y,To undo the undo']

p [put the line below the cursor]

​ c/d [number] motion

rx [replace the character at the cursor with x]

ce [chang until the end of a word]

c$ [end of line]

nG/:n [move to the NTH row in the file]

gg [move to start of file]

G [move to the bottom of file]

​ search command

自定义修改当前视图大小，用于调整终端窗口大小时不用触摸板拖拽：vscode在设置->快捷键中搜索“视图大小”

jk命令移动到较长或较短的行时，光标可能会在水平方向上左右跳转。使用gj/gk,或者直接将这两个命令映射到jk键上。

.vimrc文件

nnoremap j gj
nnoremap k gk

还是0跳到行首再jk移动吧
